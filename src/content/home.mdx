## Keycloak events to RabbitMQ publisher

The purpose of this provider is to publish Keycloak events to a RabbitMQ server via AMQP.

- [Keycloak](https://www.keycloak.org/) - It's open source identity and access management tool.
- [RabbitMQ](https://www.rabbitmq.com/) - It's message broker .
